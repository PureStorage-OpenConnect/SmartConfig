Common Icons

The following table provides information about the common icons used in the user interface.

Icon Name Description Action
In-Progress Represents an operation in-progress. None
Menu Represents any additional options. Press Menu to open a side menu with list of options.
Reachable Represents Reachable state of FlashArray. None
Not Reachable Represents Not Reachable state of FlashArray. None
Settings Represents general Settings. Press Settings to view/edit configuration settings.
Add Provides option to add FlashArray. Press Add to add FlashArray.
Close Provides option to close the current window. Press the Close icon to remove the current window, dialog box or popup message from the screen.
Trash Provides option to delete an object. Press Trash to delete an object, such as devices, images, etc.
Library Used for uploading and selecting the firmware and OS images. Press the Library icon to upload firmware and OS images.
Info Represents detailed information. Press Info to get detailed information.
Help Represents help message for the object. Press Help to view generic help message for the object.
Asterisk Indicates mandatory field to be completed. None
Progress bar Represents the progress of an operation. None
Factory Reset Provides the ability to factory reset SmartConfig. Press Reset icon to restore the SmartConfig to its original settings. Please note that any uploaded images will not be removed during this operation.
Success Represents the successful completion of an operation. None
Failure Represents the failed state of an operation. None
